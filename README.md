# 试验数据处理工具 (Tester Tool)

> 用于处理试验温控数据的桌面应用程序

## 功能特点

- 支持 CSV 数据文件解析
- Excel 数据验证与处理
- 试验稳定性判断（可配置时间间隔和温差阈值）
- 跨平台支持（Windows / macOS / Linux）
- **灵活配置**：文件索引列、通道索引列可自定义选择

## 系统要求

- macOS 10.14+（已打包为 .app，其他平台需自行打包）

## 安装

### macOS

直接双击 `TesterTool.app` 即可运行，无需安装任何依赖。

### 从源码运行

```bash
# 克隆项目
git clone https://github.com/yourusername/tester-tool.git
cd tester-tool

# 创建虚拟环境
python -m venv venv
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt

# 运行
PYTHONPATH=src python -m tester
```

## 使用方法

### 1. 选择文件

- **CSV 文件**：点击"选择 CSV 文件"按钮，选择一个或多个试验数据 CSV 文件
- **Excel 文件**：点击"选择 Excel 文件"按钮，选择 final.xlsx 模板文件

### 2. 配置参数

- **文件索引列**：选择 Excel 中哪一列作为文件索引（默认 D 列）
- **通道索引列**：选择 Excel 中哪一列作为通道索引（默认 E 列）
- **稳定时间间隔**：设置判断稳定性的时间窗口（单位：分钟，默认 60 分钟）
- **温差阈值**：设置判断稳定性的温度变化阈值（默认 2°C）

### 3. 选择环境温度

在表格中勾选作为环境温度的行（用于计算环境基准温度）

### 4. 处理数据

点击"开始处理"按钮，等待处理完成

### 5. 查看结果

处理完成后，点击"打开输出目录"按钮查看结果文件 `final-done.xlsx`

## 配置参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 文件索引列 | Excel 中用于标识文件编号的列 | D |
| 通道索引列 | Excel 中用于标识通道编号的列 | E |
| 稳定时间间隔 | 判断试验稳定性的时间窗口 | 60 分钟 |
| 温差阈值 | 允许的温度变化范围 | 2°C |

## Excel 模板格式

Excel 文件应包含以下列：

| 列 | 说明 |
|----|------|
| A | 标识 |
| B | 名称（通道名称） |
| 用户选择 | 文件索引列（如 D 列） |
| 用户选择 | 通道索引列（如 E 列） |
| I | 参考标识（填写 ref 或 -- 表示跳过） |
| J | Limit（可选，用于校验 Rise 是否超过限制） |
| F | 起始温度（输出） |
| G | 结束温度（输出） |
| H | 温差（输出） |

## 数据校验

- **F/G 列重复检查**：处理前会检查 Excel 的 F 列和 G 列是否有重复值，如有则报错
- **Rise > Limit 检查**：处理后会检查 Rise (H列) 是否超过 Limit (J列)，超过会在日志中警告

## 故障排除

### 无法打开文件

- 检查文件路径是否包含特殊字符
- 确保文件没有被其他程序打开

### 处理失败

- 检查 CSV 文件编码是否正确
- 确保 CSV 文件时间格式正确
- 确认环境温度列选择正确

### 找不到稳定时间段

- 尝试增大时间间隔或温差阈值
- 检查数据是否有足够的记录点
