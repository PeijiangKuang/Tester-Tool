# 试验数据处理工具 (Tester Tool)

> 用于处理试验温控数据的桌面应用程序

## 功能特点

- 支持 CSV 目录解析（目录包含 1/2/3/4/5 子目录，每个子目录一个 CSV 文件）
- Excel 数据验证与处理
- 试验稳定性判断（可配置时间间隔和温差阈值）
- **自动阈值递增**：初始阈值找不到稳定条件时，按步长自动递增重试
- **灵活配置**：子目录索引列、通道索引列、稳定时间间隔、温差阈值及步长均可自定义
- **实时日志**：处理过程实时输出日志，便于追踪和问题排查
- 跨平台支持（Windows / macOS / Linux）

## 系统要求

- macOS 10.14+（已打包为 .app，其他平台需自行打包）

## 安装

### macOS

直接双击 `TesterTool.app` 即可运行，无需安装任何依赖。

### 从源码运行

```bash
# 克隆项目
git clone https://github.com/yourusername/tester-tool.git
cd tester-tool

# 创建虚拟环境
python -m venv venv
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt

# 运行
PYTHONPATH=src python -m tester
```

## 使用方法

### 1. 选择 CSV 目录

- 点击"选择 CSV 目录"按钮，选择包含 1/2/3/4/5 子目录的目录
- 每个子目录需包含一个 CSV 文件，子目录名必须为数字

### 2. 选择 Excel 文件

- 点击"选择 Excel 文件"按钮，选择 final.xlsx 模板文件

### 3. 配置参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 子目录索引列 | 对应子目录名在 Excel 中的列（默认 D 列） | D |
| 通道索引列 | Excel 中用于标识通道编号的列（默认 E 列） | E |
| 稳定时间间隔 | 判断稳定性的时间窗口（单位：分钟） | 60 |
| 温差阈值（初始值） | 首次判断稳定性的温度变化阈值（单位：K） | 2 |
| 温差阈值步长 | 找不到稳定条件时，每次递增的步长（单位：K） | 1 |

### 4. 选择环境温度

在表格中勾选作为环境温度的行（用于计算环境基准温度）

### 5. 处理数据

- 点击"开始处理"按钮
- 实时日志会显示处理进度和详细信息
- 如果初始阈值找不到稳定条件，系统会自动按步长递增阈值重试

### 6. 查看结果

处理完成后，点击"打开输出目录"按钮查看结果文件 `final-done.xlsx`

## Excel 输出格式

| 列 | 说明 |
|----|------|
| A | 标识 |
名称（通道名称| B | ） |
| 用户选择 | 子目录索引列（如 D 列） |
| 用户选择 | 通道索引列（如 E 列） |
| I | 参考标识（填写 ref 或 -- 表示跳过） |
| J | Limit（温升限值） |
| F | 起始温度（输出） |
| G | 结束温度（输出） |
| H | 温差/温升（输出） |

### 输出样式规则

- **环境温度行**：H 列填写 `-`
- **负数填充**：H 列为负数时，单元格填充黄色背景
- **超限标红**：温升（H列）超过限值（J列）时，单元格文字标红

## 错误提示

工具提供分阶段错误提示，便于定位问题：

1. **目录检查**：验证目录结构是否正确（1/2/3/4/5 子目录）
2. **CSV 解析**：解析各子目录中的 CSV 文件
3. **文件校验**：检查 Excel 文件格式、重复值等
4. **数据处理**：处理过程中的详细日志

## 故障排除

### 无法打开文件

- 检查文件路径是否包含特殊字符
- 确保文件没有被其他程序打开

### 处理失败

- 检查 CSV 文件编码是否正确
- 确保 CSV 文件时间格式正确
- 确认环境温度列选择正确

### 找不到稳定时间段

- 尝试增大初始温差阈值或步长
- 检查数据是否有足够的记录点
