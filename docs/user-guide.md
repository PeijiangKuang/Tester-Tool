# 试验数据处理工具 - 用户指南

## 简介

试验数据处理工具是一款用于处理温控试验数据的桌面应用程序。

## 功能特性

- **CSV 数据解析**：支持解析多通道温度传感器数据
- **Excel 模板处理**：读取和写入 Excel 格式的模板文件
- **稳定性判断**：自动判断试验是否达到稳定状态
- **灵活配置**：可自定义文件索引列和通道索引列
- **数据校验**：检查 Rise 是否超过 Limit
- **跨平台**：支持 Windows、macOS、Linux 系统

## 快速开始

### 1. 选择文件

1. 点击"选择 CSV 文件"按钮，选中一个或多个 CSV 数据文件
2. 点击"选择 Excel 文件"按钮，选择包含模板的 Excel 文件（final.xlsx）

### 2. 配置参数

根据你的 Excel 文件格式，配置以下参数：

- **文件索引列**：Excel 中哪一列是文件编号（如 D 列）
- **通道索引列**：Excel 中哪一列是通道编号（如 E 列）
- **稳定时间间隔**：判断稳定性的时间窗口（默认 60 分钟）
- **温差阈值**：判断稳定性的温度变化阈值（默认 2°C）

### 3. 选择环境温度

在表格中勾选用于计算环境基准温度的行（这些行的温度将作为环境温度参考）

### 4. 开始处理

点击"开始处理"按钮，等待处理完成

### 5. 查看结果

处理完成后，点击"打开输出目录"按钮，在原目录找到 `final-done.xlsx` 文件

## 参数详解

### 稳定时间间隔

指判断试验稳定性的时间窗口长度。例如选择"60分钟"，表示检查任意1小时间隔内温度是否稳定。

### 温差阈值

指允许的温度变化范围。默认值为 2°C，表示在时间间隔内，温度变化不超过 2°C 视为稳定。

### 文件索引列 / 通道索引列

Excel 中用于关联 CSV 数据的列：
- 文件索引列：标识数据来源于哪个 CSV 文件
- 通道索引列：标识数据来自哪个温度通道

## 数据格式

### CSV 文件格式

CSV 文件应包含：
- 表头行（包含"扫描"关键字）
- 时间列（格式：YYYY/M/D H:M:S:ms）
- 温度列（格式：N (C)，其中 N 为通道号）

### Excel 模板格式

Excel 文件应包含以下列：

| 列 | 说明 |
|----|------|
| A | 标识 |
| B | 名称（通道名称） |
| D | 文件编号（可配置） |
| E | 通道编号（可配置） |
| I | 参考标识（ref/-- 表示跳过） |
| J | Limit（可选） |
| F | 起始温度（输出） |
| G | 结束温度（输出） |
| H | 温差（输出） |

## 输出说明

处理完成后，Excel 文件将：
- 重命名为 `final-done.xlsx`
- 填写 F、G、H 列数据
- 更新表头时间信息

如果在处理过程中发现 Rise > Limit 的情况，会在日志中显示警告信息。

## 常见问题

### Q: 程序打不开怎么办？
A: macOS 可能需要右键选择"打开"，或在终端中授予执行权限。

### Q: 选择 Excel 后程序闪退？
A: 请检查 Excel 文件格式是否符合模板要求，或查看日志区域的错误信息。

### Q: 找不到稳定时间段？
A: 尝试增大"稳定时间间隔"或"温差阈值"参数。

### Q: 如何修改索引列？
A: 在"参数设置"区域修改"文件索引列"和"通道索引列"的下拉选项。
